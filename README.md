## Build and Run 

### Building JUnit 4 
* Checkout the git tag `junit4-startpoint`
```bash 
git checkout junit4-startpoint
```
* Run the tests as a sanity check 
```bash
./mvnw clean test
```

### Building JUnit 5
* Checkout the git tag 
```bash 
git checkout <in-progress>
```
* Run the tests as a sanity check 
```bash
./mvnw clean test
```

### Running The App 
* Run the following command 
```bash
./mvnw spring-boot:run 
```
